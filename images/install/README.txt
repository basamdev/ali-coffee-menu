Add to Home Screen tutorial images
====================================

Put your screenshot PNG or JPG files here with these exact names:

iPhone (iOS):
  ios-step-1.png   (tap browser menu …)
  ios-step-2.png   (tap Share)
  ios-step-3.png   (tap Add to Home Screen)
  ios-step-4.png   (tap Add)

Android:
  android-step-1.png   (tap Menu ⋮ in Chrome)
  android-step-2.png   (choose Add to Home screen)
  android-step-3.png   (choose Install)
  android-step-4.png   (confirm Install)

The menu shows them when the user taps:
Info button → "زیادکردن بۆ سکرین" / Add to Home Screen

No popup on admin or other pages — only this button on menu.
